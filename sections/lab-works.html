ЛАБОРАТОРИЯЛЫК ИШ №1 (7-класс)

🔬 ФИЗИКА БӨЛҮМҮ

Тема: Катуу нерсенин өлчөмдөрүн өлчөө жана көлөмүн аныктоо

🎯 Иштин максаты:

  • ✔ Сызгыч жана штангенциркуль менен өлчөөлөрдү жүргүзүүнү үйрөнүү;
  • ✔ Нерсенин узундугун, туурасын жана бийиктигин аныктоо;
  • ✔ Өлчөө жыйынтыктары боюнча нерсенин көлөмүн эсептөө.

🧰 Керектүү жабдуулар:

  • Сызгыч
  • Штангенциркуль
  • Жыгач же металл брусок
  • Карандаш жана дептер

🖼️ Эксперименттин схемасы:

Узундугу (a) ┌───────────────────┐ / /│ / / │ └───────────────────┘ │ │ │ │ ↑ │ │ │ │ Бийиктиги (h) │ │ │ ↓ │ │ / └───────────────────┘ Туурасы (b)

📋 Иштин жүрүшү:

1️⃣ Брусканын узундугун (a) сызгыч менен өлчөңүз.
2️⃣ Анын туурасын (b) өлчөңүз.
3️⃣ Бийиктигин (h) аныктаңыз.
4️⃣ Өлчөө жыйынтыктарын төмөнкү таблицага жазыңыз.
5️⃣ Көлөмүн формула боюнча эсептеңиз: V = a × b × h

📊 Өлчөөлөрдүн таблицасы:

* Төмөнкү уячаларга өзүңүздүн өлчөөлөрүңүздү жазыңыз:
