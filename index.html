# 🌟 Mugalim.AI — Заманбап Физика жана Астрофизика Порталы **Mugalim.AI** — бул физика мугалимдери, окуучулар жана илимге кызыккандар үчүн иштелип чыккан кыргыз тилдүү инновациялык платформа. --- ## 📂 Сайттын бөлүмдөрү: * **🎮 Тесттик оюндар** — Физикалык кубулуштарды оюн түрүндө үйрөнүү. * **📝 Тесттер** — Билимди текшерүү жана деңгээлди аныктоо. * **📚 Сабактын пландары** — Мугалимдер үчүн даяр методикалык колдонмолор. * **🌌 Физика жана Астрофизика** — Ааламдын сырлары жана терең илимий макалалар. * **🧪 Физика, лабораториялык иштер** — Виртуалдык симуляциялар жана тажрыйбалар. * **👤 Автор боюнча маалымат** — Долбоордун негиздөөчүсү жана максаты жөнүндө. --- ## 🛠 Техникалык стек: - HTML5, CSS3 (Tailwind CSS) - JavaScript (Interactive Simulations) - GitHub Pages (Hosting) --- ## 👨‍🏫 Автор жөнүндө Бул бөлүмдө автордун тажрыйбасы, билим берүү тармагындагы жетишкендиктери жана байланыш маалыматтары орун алат.
